Seu objetivo é criar um sistema para gerenciar invoices, 

precisa ter cadastro de email e password para logar, e cada user vai ter o seu perfil.

O nome do app se chama MYNVOICE, pode usar esse nome no sistema.

Não faça um sistema no padrão standard que todos estão acostumados de criar um sistema, faça algo diferente, algo que chame a atenção, algo que seja bonito, algo que seja diferente.

o objetivo do sistema é organizar a vida de pequenas empresas, self employed e freelancers. Aonde precisam ter um minimo de gerenciamento de invoices e despesas.

O processo de criar invoice deve ser facil, pratico e rapido.

O sitema precisa ter alguns tipos de cadastros

da minha empresa ou self employed
de empresas para quem vou emitir a invoice
ter um dashboard com as pagas e não pagadas e alguns dados e graficos
ter uma tela gerencial para ver as invoices e se foram pagas
ter a opção de colocar as despesas
colocar data de pagamento da invoice e por qual meio que foi pago (cartao, transferencia e etc..) para dar baixa

Ter um cadastro para as despesas fixas e variadas, por grupo de despesas.


ter a opção de ja mandar a invoice pelo sistema para a empresa que vai receber (por email)

Algumas telas na pasta /examples para ter ideias de telas, não é para copiar, é para fazer algo melhor

um sitema muito bom é o invoice.zoho.eu pode se basear neles
